**Global Configurations**

[See here][1] for more info

```sh
$ git config --global user.name "<YOUR NAME>"
$ git config --global user.email <YOUR EMAIL>
```

**Cloning the Repo**

The repo is hosted [on github][2]

```sh
$ git clone https://github.com/kescobo/KCLabgit.git
$ cd KCLabgit
```
